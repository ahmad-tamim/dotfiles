| `Cmd+Shift+P` | Command Palette | Show all commands |
| `Cmd+Shift+O` | Go to Symbol | Navigate to symbol in file |
| `Cmd+T` | Go to Symbol (workspace) | Search symbols across workspace |
| `Cmd+G` | Go to Line | Jump to specific line number |

## Tips

- **Chord Shortcuts**: Shortcuts like `Cmd+K Cmd+E` are "chords" - press `Cmd+K`, release, then `Cmd+E`
- **Multi-cursor Power**: Use `Cmd+D` repeatedly to select multiple instances, then edit them all at once
- **Quick File Navigation**: `Cmd+P` then type filename to quickly open files
- **Symbol Search**: Use `@` in Quick Open (`Cmd+P`) to search symbols in current file
- **Command Palette**: `Cmd+Shift+P` gives access to ALL VSCode commands
- **Zen Mode**: `Cmd+K Z` for distraction-free coding (press `Esc Esc` to exit)

---

**Remember**: You can always press `Cmd+K Cmd+S` to see all keyboard shortcuts in VSCode!
